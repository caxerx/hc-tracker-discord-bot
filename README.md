# HC Tracker Discord Bot
A discord bot for internal usage. It records the completion of Hardcore Raids in Nostale and generates reports to count the completions for each character to find out the characters that do not reach the completion target and reward the characters that have the most completions.

## Features

### Raid Tracking System
- Record and track daily raid completions for each character
- Generate completion reports to identify performance
- Image evidence submission for raid completion verification

### Raid Event Management
- Create and manage Discord scheduled events for coordinating raids
- Public event creation available to all members via `/createraid` command
- Automatic notifications to designated RaidEvent channels
- Integration with Discord's native event system
